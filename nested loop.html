HTML tags

  1. Anchor Tag (a)
    2. Defines a hyperlink
  2. Abbrevation Tag (abbr)
    Defines an abbreviation or an acronym
  3. Bold tag (b)
    Defines bold text
  4. Line break tag (br)
    Defines a single line break
  5. Button tag (button)
    Defines a clickable button
  6. Center tag (center)
    Defines centered text
  7. Quotation Tag(q)
    Defines text in b/w double quotes
  8. Font tag(font)
    Defines font, color, and size for text
  9. Horizontal ruler tag(hr)
    Defines a thematic change in the content
  10. Paragraph tag(p)
    Defines a paragraph
  11. Preformatting tag(pre)
    Defines preformatted text
  12. Image tag (img)
    Defines an image
  13. Heading tag(h1-h6)
    Defines HTML headings
  14. Mark tag(mark)
    Defines marked/highlighted text
  15. Strike tag(strike)
    Defines strikethrough text
  16. Subscript tag(sub)
    Defines subscripted text
  17. Superscript tag(sup)
    Defines superscripted text
  18. Video tag(video)
    Defines embedded video content
